## How to Run Locally (not using Docker)

1. Clone the repo
2. `cd nestjs-firebase`
3. `npm i`
4. Enter environment variables using `touch .env` based on `.env.example`
5. Run PostgreSQL DB
6. Run the app with `npm run start:dev`

## How to Run Locally (using Docker)

1. Clone the repo
2. `cd nestjs-firebase`
3. `docker compose up`
